# 数据库
数据库定义看 model, 如有特殊的需要解释的数据库model 请看 database.md


# 运行

推荐使用webstorm

1. npm install
2. 在model/sync/下执行 所有的　init-*.js (node init-gym.js ...等等)
3. node app.js 或者 调试debug webstorm


# 项目结构

- app.js            主入口
- app               项目主目录
    - control           控制层
    - model             数据库model层
    - route-list        路由层
        - middleware        中间件
    - unit              工具件
    - route.js          路由入口文件
- conf              配置文件


# 接口定义
总体形式为
```
{
    code: 1     // 只有1为请求成功 其他值需要额外处理
    msg:  'ok'  // 描述信息
    data: {} // 具体data
}


```
